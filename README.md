# JankyXML-to-UDDF

Tool to convert the exported data from the VMS Redbare into UDDF so that it can be worked by most diving applications.

## Diving Specifics

* [VMS Redbare](http://www.vmsrebreathers.com/)
* [Nammu Tech](https://www.nammu-tech.io/)
* [UDDF](https://www.streit.cc/extern/uddf_v321/en/index.html)
